# 书搜搜 — UI设计规范

## 版本历史
| 版本 | 日期 | 修改内容 |
|------|------|---------|
| v1.0 | 2025-06-10 | 初版 |

---

## 1. 整体风格
- 简洁、清晰、书籍主题
- 主色调：暖色系（米白/浅灰背景，棕色/橙色点缀）
- 字体：系统默认字体（无特殊要求）
- 中英双语界面，右上角切换按钮

## 2. 导航栏布局
`
┌──────────────────────────────────────────────────────────┐
│  📚 书搜搜 / BookSearch    🔍 [搜索框]                  │
│  首页 | 论坛 | 交换 | [用户名] | [EN/中文]               │
└──────────────────────────────────────────────────────────┘
`

## 3. 搜索结果卡片样式

### 3.1 卡片布局
`
┌──────────────────────────────────────────────────────────┐
│  📘 书名 - 作者                                         │
│  分类：计算机/人工智能  |  ISBN：978-7-302-xxx           │
│  ──────────────────────────────────────────────          │
│  🏛️ 图书馆：  状态标签  + 详细信息                        │
│  💬 书友评论：张三说"..."  👍12                          │
│  🔄 交换信息：有书友愿意交换 → [联系TA]                  │
│  ──────────────────────────────────────────────          │
│  💡 AI推荐理由：这本书是入门机器学习的经典之作...         │
└──────────────────────────────────────────────────────────┘
`

### 3.2 状态标签样式
| 状态 | 标签颜色 | 文字 |
|------|---------|------|
| 在馆 | 绿色背景 | ✅ 在馆（3楼A区05架） |
| 已借出 | 黄色背景 | ⏳ 已借出，预计X月X日归还 |
| 无匹配 | 红色背景 | 📝 未找到，可向图书馆推荐 |
| 可交换 | 蓝色背景 | 🔄 有书友可交换 |

### 3.3 评论位置
评论直接**接在书籍信息后面**，而不是单独一行：
`
📘 书名
  图书馆：✅ 在馆
  书友评论1：...
  书友评论2：...
  ─────────────────
📘 另一本书
  ...
`

## 4. 页面列表

| 页面 | URL | 说明 |
|------|-----|------|
| 首页 | / | 网站介绍 + 搜索入口 |
| 搜索 | /search | 搜索框 + 结果展示 |
| 论坛列表 | /forum | 帖子列表 |
| 帖子详情 | /forum/{id} | 帖子内容 + 评论 |
| 发帖 | /forum/new | 发帖表单 |
| 交换列表 | /exchange | 可借书籍列表 |
| 发布图书 | /exchange/publish | 发布表单 |
| 借阅沟通 | /exchange/{id}/chat | 站内临时评论区 |
| 登录 | /auth/login | 登录页 |
| 注册 | /auth/register | 注册页 |
| 个人主页 | /profile | 用户信息 |

## 5. 响应式设计
- 电脑端：卡片横向排列（一行2~3个）
- 手机端：卡片竖向排列（一行1个）
- 导航栏自动折叠（手机上）

## 6. 中英双语切换
- 右上角提供语言切换按钮
- 默认中文
- 所有文案支持中英文切换
- 切换后刷新页面生效（简单实现）
