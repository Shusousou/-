| 文件 | 说明 |
|-----|------|
| 日志使用说明.md | 如何写日志 |
| YYYY-MM-DD.md | 每日开发日志 |
| 模板.md | 日志模板 |

## 工作流程说明
1. 每天开始开发前，查看前一天的日志和今日计划
2. 开发中遇到问题，记录到日志
3. 每天结束前，更新日志
4. 重大决策更新到对应文档

## 各文档文件路径
- 项目文档：docs/ 目录下
- 开发日志：logs/ 目录下
- 项目源码：shusousou/ 目录下
- 配置文件：shusousou/config.py
- 主入口：shusousou/main.py
