## 1. 表结构总览

| 表名 | 说明 | 所属模块 |
|------|------|---------|

## 2. 详细表结构

### 2.1 users 用户表
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| email | VARCHAR(100) | NOT NULL, UNIQUE | 邮箱 |
| password_hash | VARCHAR(200) | NOT NULL | 密码哈希值 |
| is_verified | BOOLEAN | DEFAULT FALSE | 是否已验证邮箱 |
| created_at | TIMESTAMP | DEFAULT NOW | 注册时间 |

### 2.2 posts 论坛帖子表
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO_INCREMENT | 帖子ID |
| user_id | INTEGER | FK → users.id | 发帖用户 |
| book_name | VARCHAR(200) | NOT NULL | 书名 |
| author | VARCHAR(100) | | 作者 |
| isbn | VARCHAR(20) | | ISBN（关键关联字段） |
| category | VARCHAR(50) | | 分类 |
| content | TEXT | NOT NULL | 评论内容 |
| likes_count | INTEGER | DEFAULT 0 | 点赞数 |
| created_at | TIMESTAMP | DEFAULT NOW | 发帖时间 |

### 2.3 comments 评论表
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO_INCREMENT | 评论ID |
| post_id | INTEGER | FK → posts.id | 所属帖子 |
| user_id | INTEGER | FK → users.id | 评论用户 |
| content | TEXT | NOT NULL | 评论内容 |
| created_at | TIMESTAMP | DEFAULT NOW | 评论时间 |

### 2.4 likes 点赞表
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO_INCREMENT | |
| post_id | INTEGER | FK → posts.id | 被点赞的帖子 |
| user_id | INTEGER | FK → users.id | 点赞用户 |
| UNIQUE(post_id, user_id) | | | 一个用户只能点赞一次 |

### 2.5 exchange_books 交换图书表
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO_INCREMENT | 图书ID |

| author | VARCHAR(100) | | 作者 |
| isbn | VARCHAR(20) | | ISBN |
| requirements | TEXT | | 借阅要求 |
| expectations | TEXT | | 期望（讨论/读后感等） |
| status | VARCHAR(20) | DEFAULT 'available' | 状态：available/borrowed |
| created_at | TIMESTAMP | DEFAULT NOW | 发布时间 |

### 2.6 exchange_requests 交换Request表
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO_INCREMENT | |
| user_id | INTEGER | FK → users.id | 发起人 |
| description | TEXT | NOT NULL | 需求描述 |
| keywords | TEXT | | AI提取的关键词（JSON） |
| is_active | BOOLEAN | DEFAULT TRUE | 是否有效 |
| created_at | TIMESTAMP | DEFAULT NOW | |

### 2.7 exchange_messages 站内临时消息表
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK, AUTO_INCREMENT | |
| book_id | INTEGER | FK → exchange_books.id | 关联的图书 |
| sender_id | INTEGER | FK → users.id | 发送方 |
| receiver_id | INTEGER | FK → users.id | 接收方 |

| created_at | TIMESTAMP | DEFAULT NOW | |

## 3. 数据关联示意图

`
users ────────── posts (发帖)
users ────────── comments (评论)
users ────────── likes (点赞)
users ────────── exchange_books (发布交换图书)
users ────────── exchange_requests (挂Request)

posts ────────── comments (帖子下的评论)
posts ────────── likes (帖子的点赞)

exchange_books ── exchange_messages (图书下的沟通)
exchange_books ── users (owner_id 关联放书人)
`

## 4. 关键关联字段

### ISBN
- **作用**：连接图书馆API、论坛、交换的核心字段
- **出现位置**：posts 表、exchange_books 表
- **使用方式**：通过ISBN查询图书馆API获取馆藏状态

### user_id
- **作用**：所有模块通过用户ID关联到用户系统
- **出现位置**：所有包含用户操作的表中
