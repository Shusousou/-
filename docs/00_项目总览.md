# 书搜搜网站 — 项目总览

## 项目简介
书搜搜是一个面向校园的智能图书服务平台，集成了AI图书推荐、书籍交换借阅和好书推荐论坛三大功能。

## 项目目标
1. 帮助用户通过AI快速找到想读的书
2. 搭建P2P书籍交换平台，促进图书流转
3. 建立读书社区，分享阅读体验

## 项目定位
- 目标用户：在校学生
- 开发阶段：MVP（最小可行产品）
- 技术路线：Python全栈 + AI辅助

## 核心功能一览
| 模块 | 功能 | 状态 |
|------|------|------|
| ① 书搜搜 | AI生成推荐 + 图书馆数据查询 | 待开发 |
| ② 论坛 | 发帖、回帖、点赞、分类搜索 | 待开发 |
| ③ 交换借阅 | 发布/借阅/Request匹配 | 待开发 |
